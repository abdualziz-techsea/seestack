# Arabic RTL Language Switch

## Steps
1. Navigate to /overview
2. Verify page direction is "ltr"
3. Verify font is Geist (English)
4. Click "EN" language toggle button in topbar
5. Verify button now shows "AR"
6. Verify document direction changes to "rtl"
7. Verify sidebar moves to right side of screen
8. Verify navigation labels show Arabic text
9. Verify font changes to Rubik
10. Verify code blocks and timestamps remain LTR
11. Click "AR" toggle — verify switches back to English LTR
12. Reload page — verify language preference persists from localStorage
