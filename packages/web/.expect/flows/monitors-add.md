# Add Monitor Flow

## Steps
1. Navigate to /monitors
2. Verify monitor grid is visible
3. Click "+ Add monitor" button — verify modal opens
4. Leave URL empty, click Add — verify validation error
5. Enter URL "https://api.myapp.com/health"
6. Enter name "API Health Check"
7. Select interval "5 min"
8. Enter threshold 2000
9. Click Add monitor — verify modal closes
10. Verify new monitor card appears in grid
11. Verify status dot is visible on new card
