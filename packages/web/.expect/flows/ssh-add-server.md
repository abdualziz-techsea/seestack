# Add SSH Server Flow

## Steps
1. Navigate to /ssh
2. Click "+ Add server" button — verify sheet slides open
3. Enter server name "Production API"
4. Enter host "10.0.1.50"
5. Enter port "22"
6. Enter username "deploy"
7. Click "Test Connection" — verify connection test result
8. Click "Add Server" — verify sheet closes
9. Verify new server card appears in the list
